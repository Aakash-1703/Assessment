Sales Analysis Dashboard

Project Overview:

This project presents a comprehensive analysis of retail sales performance using data-driven insights and visual storytelling. The goal is to identify key trends, customer behavior patterns, and revenue drivers to support informed business decision-making.
The project combines analytical workflows in a Google Colab with a visually rich dashboard for effective communication of insights.
